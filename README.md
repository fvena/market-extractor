# market-extractor

Library for scraping and aggregating data from global stock markets

## Installation

```bash
npm install market-extractor
```

## Usage

```typescript
import { add } from "market-extractor";

console.log(add(1, 2)); // 3
```

## Development

- `npm run dev`: Start development mode with auto-rebuild
- `npm run build`: Build the library
- `npm run start`: Start the library
